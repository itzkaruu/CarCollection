# CarCollection
A personal car showcase web app where you can add, browse, and manage your favorite cars — built with HTML, CSS, and JavaScript.

🚗 Carl's Car Collection
A personal car showcase website built with vanilla HTML, CSS, and JavaScript. This project allows users to build and manage their own car collection entirely from the browser — no backend required.
Features

Browse a collection of cars with detailed specs
Add new cars with custom info, image, and YouTube review link
Remove cars with a confirmation prompt
Filter cars by category (Sports, SUV, Hypercar, Electric, Pickup, Sedan)
Search cars by brand, name, or description
Click any car to watch a YouTube review in a new tab
All data persists using localStorage
Responsive design that works on mobile and desktop

Built With

HTML
CSS
JavaScript (Vanilla)
localStorage for data persistence

How to Run
Clone the repo and open home.html with Live Server in VS Code or just open it directly in your browser.
